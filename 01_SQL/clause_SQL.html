1 - Les requetes de base

  • Créer une BDD : CREATE DATABASE IF NOT EXIST nomBdd
  • Pour supprimer une BDD : DROP DATABASE IF EXIST nomBdd
  • Observer la description d'une BDD : DESC nomTable
  • Voir les BDD dans le server : SHOW DATABASES

2 - Tables et colonnes

  • Creer une BDD 'entreprise' : CREATE DATABASE IF NOT EXISTS entreprise

  • Créer une table dans la base de donnée entreprise : 
                                        
                                        
                                            CREATE TABLE IF NOT EXISTS employes (
                                            id_emloyé INT (11) NOT NULL PRIMARY KEY AUTO_INCREMENT, 
                                            prenom VARCHAR (20) NULL, 
                                            nom VARCHAR (20) NULL, 
                                            sexe ENUM ('f', 'm') NOT NULL, 
                                            service VARCHAR (30) NULL,
                                            date_embauche DATE NULL, 
                                            salaire FLOAT NULL )

3 - SELECT & DISTINCT

  • Pour voir tous le contenu d'une table : SELECT * FROM nomTable
  • Selectionner les valeurs de deux champs d'une table: SELECT colonne1, colonne2 FROM nomTable
    -> SELECT prenom, nom, FROM employe
  • Afficher les données sans doublons : SELECT DISTINCT (colonne) FROM nomTable
    -> SELECT DISTINCT service FROM employes
  • Effacer un enregistrement : DELETE FROM nomTable WHERE colonne = " " DELETE FROM employes WHERE prenom = 'Salimata'
